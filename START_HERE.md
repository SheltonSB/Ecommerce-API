## 🎯 **TO RUN THE API (Choose ONE method):**

### **Method 1: Batch File (Easiest)**
1. Open File Explorer
2. Navigate to project folder
3. **Double-click** `run-api.bat`
4. Wait for "Now listening on: http://localhost:5154"
5. Open browser to **http://localhost:5154**

### **Method 2: Terminal**
1. Open PowerShell in project folder
2. Run:
```powershell
cd Ecommerce.Api
dotnet run
```
3. Wait for "Now listening on: http://localhost:5154"
4. Open browser to **http://localhost:5154**

### **Method 3: Visual Studio**
1. Open `Ecommerce.sln`
2. Press **F5** or click green "Run" button
3. Browser opens automatically to Swagger UI
